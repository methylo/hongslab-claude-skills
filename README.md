# HONGS LAB Claude Skills

홍작가(홍순성)가 1인 기업·콘텐츠 제작·강의용으로 설계한 Claude 스킬 모음입니다. 글쓰기, 책쓰기, 강의 자료, 리서치 등의 작업을 자동화합니다.

## 다운로드 방법

### 방법 1. ZIP 전체 다운로드 (권장)

1. 이 페이지 상단의 초록색 `Code` 버튼 클릭
2. `Download ZIP` 클릭
3. 압축 해제

### 방법 2. 개별 스킬 폴더 다운로드

원하는 스킬 폴더 클릭 → 파일 화면에서 `Download raw file` 아이콘 클릭

## 사용 방법

1. 압축 해제한 스킬 폴더를 Claude 채팅창에 업로드합니다.
2. 트리거 키워드로 호출합니다.

```
예시
"글쓰기 스킬로 작성해줘"
"write-content 스킬로 정리해줘"
```

각 스킬의 트리거 키워드는 해당 폴더의 `SKILL.md` 파일 상단에서 확인할 수 있습니다.

## 포함된 스킬

### 글쓰기

| 스킬명 | 용도 |
|--------|------|
| `write-content` | 단편 콘텐츠 작성 (칼럼·뉴스레터·보고서 등) |

스킬은 점진적으로 추가될 예정입니다.

## 스킬이란

Claude Skills는 특정 작업의 워크플로우를 코드화한 지시문 모음입니다. 매번 같은 작업을 설명할 필요 없이, 스킬 폴더를 업로드하고 트리거 키워드로 호출하면 동일한 품질의 결과를 반복적으로 얻을 수 있습니다.
